# Web APIs (Expanded)

Web APIs are built-in browser features that allow JavaScript to interact with the browser and device capabilities.

They are not part of core JavaScript, but are provided by the browser environment.

---

# Why Web APIs are Important

Web APIs allow websites to:

- Store data in the browser
- Make network requests
- Control browser history
- Access device features
- Handle multimedia
- Create interactive applications

---

# Categories of Web APIs

## 1. Storage APIs

### Local Storage

Stores data in the browser permanently (until manually cleared).

```js
localStorage.setItem("name", "Navaneeth");
console.log(localStorage.getItem("name"));
```

### Session Storage

Stores data only for the current session.

```js
sessionStorage.setItem("theme", "dark");
```

---

## 2. Fetch API (Networking)

Used to make HTTP requests.

```js
fetch("https://api.example.com/data")
    .then(response => response.json())
    .then(data => console.log(data));
```

---

## 3. DOM API

Allows JavaScript to manipulate HTML elements.

```js
document.getElementById("title").innerText = "Hello World";
```

---

## 4. Timer APIs

Used for delays and repeated execution.

```js
setTimeout(() => {
    console.log("Executed after 2 seconds");
}, 2000);

setInterval(() => {
    console.log("Runs every 1 second");
}, 1000);
```

---

## 5. Geolocation API

Gets user location (with permission).

```js
navigator.geolocation.getCurrentPosition(position => {
    console.log(position.coords.latitude);
    console.log(position.coords.longitude);
});
```

---

## 6. Clipboard API

Used to copy or read clipboard data.

```js
navigator.clipboard.writeText("Hello World");
```

---

## 7. History API

Controls browser navigation history.

```js
history.back();
history.forward();
```

---

## 8. Canvas API

Used to draw graphics on the web.

```html
<canvas id="myCanvas"></canvas>

<script>
const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");

ctx.fillStyle = "red";
ctx.fillRect(10, 10, 100, 100);
</script>
```

---

# Web API Flow

1. Browser provides API
2. JavaScript calls API
3. Browser executes action
4. Result is returned to JS

---

# Example: Simple API Usage Together

```html
<button onclick="getData()">Load Data</button>

<script>
function getData() {
    fetch("https://jsonplaceholder.typicode.com/posts/1")
        .then(res => res.json())
        .then(data => {
            console.log(data);
        });
}
</script>
```

---

# Summary

- Web APIs are browser-provided tools for JavaScript.
- They extend JavaScript capabilities.
- Common APIs include Storage, Fetch, DOM, Geolocation, and Canvas.
