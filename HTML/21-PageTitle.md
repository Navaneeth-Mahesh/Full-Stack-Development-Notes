# HTML Practice Page

This practice page combines the HTML concepts you've learned so far into a single webpage. It is designed to help you understand how different HTML elements work together in a real document.

## Topics Covered

- HTML Document Structure
- Headings
- Paragraphs
- Line Breaks (`<br>`)
- Horizontal Rules (`<hr>`)
- Links
- Images
- Text Formatting
- Preformatted Text
- Quotation Elements
- Colors (Inline CSS)
- Buttons
- HTML Comments

---

## Learning Objectives

After completing this practice page, you should be able to:

- Create a complete HTML document.
- Use common HTML elements correctly.
- Add links and images to a webpage.
- Format text using HTML tags.
- Apply basic inline CSS for colors and borders.
- Write clean and well-organized HTML code.
- Understand how multiple HTML elements work together to build a webpage.

---

## Practice Tips

- Type the code yourself instead of copying and pasting.
- Modify the text, colors, and images to see how the output changes.
- Experiment with different HTML elements and attributes.
- Open the page using **Live Server** to view changes instantly.
- Read the code comments—they explain the purpose of each section.

---

## Suggested Exercises

Try completing these exercises on your own:

1. Change the webpage title.
2. Replace the heading with your own name.
3. Add another paragraph about yourself.
4. Insert your own image.
5. Create a link to your favorite website.
6. Make an image clickable.
7. Add another quotation using the `<q>` tag.
8. Change the background and text colors using the `style` attribute.
9. Create a button that opens another webpage.
10. Add comments to organize different sections of the page.

---

## Challenge

Without referring to your notes, create a new HTML page that includes:

- A proper HTML document structure
- At least two headings
- Three paragraphs
- A hyperlink
- An image
- Five text formatting elements
- One blockquote
- One abbreviation
- One address
- One button
- Different colors using inline CSS

If you can complete this challenge successfully, you've built a solid foundation in the basics of HTML.
