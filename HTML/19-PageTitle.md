# HTML Practice Page

This practice page combines the HTML concepts you've learned into one complete webpage. It helps you understand how different HTML elements work together to create a real webpage.

Instead of learning tags individually, you'll practice using them together in a meaningful project.

---

# Topics Covered

This practice page includes the following HTML concepts:

* HTML Document Structure
* Headings (`<h1>`–`<h6>`)
* Paragraphs (`<p>`)
* Line Breaks (`<br>`)
* Horizontal Rules (`<hr>`)
* Links (`<a>`)
* Images (`<img>`)
* Text Formatting Elements
* Preformatted Text (`<pre>`)
* HTML Quotation Elements
* HTML Colors (Inline CSS)
* Buttons (`<button>`)
* HTML Comments
* Basic Inline CSS using the `style` attribute

---

# Learning Objectives

After completing this practice page, you should be able to:

* Create a complete HTML document from scratch.
* Understand the purpose of common HTML elements.
* Use headings and paragraphs to organize content.
* Insert links and images into a webpage.
* Format text using semantic HTML elements.
* Add quotations, abbreviations, and contact information.
* Apply simple styling using the `style` attribute.
* Organize HTML code using comments.
* Build a simple webpage using everything you've learned so far.

---

# Practice Tips

To get the most out of this exercise:

* **Type the code yourself** instead of copying and pasting.
* Experiment by changing text, colors, images, and links.
* Save your file frequently and refresh the browser to see the changes.
* Use **Live Server** in Visual Studio Code for automatic page refresh.
* Read every HTML comment to understand why each section exists.
* Try removing or modifying tags to see how the webpage changes.
* Make mistakes—they're one of the best ways to learn HTML.

# Common Beginner Mistakes

❌ Forgetting to close HTML tags.

```html
<h1>Welcome
```

✔ Correct

```html
<h1>Welcome</h1>
```

---

❌ Using headings only to make text bigger.

Use headings based on their importance, not their appearance.

---

❌ Using multiple `<br>` tags to create spacing.

Use CSS for spacing instead of repeated line breaks.

---

❌ Copying code without understanding it.

Always type the code yourself and experiment with it.
