# HTML (HyperText Markup Language)

HTML stands for **HyperText Markup Language**.

* HTML was invented by Sir Tim Berners-Lee
* HTML is used to create web pages.
* It defines the structure of a web page.
* HTML is made up of **elements**.
* These elements tell the browser how to display content.
* Elements can represent headings, paragraphs, links, images, tables, and more.

## Basic HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>

<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>

<script>
    // JavaScript code can be written here
</script>

</html>

## How to Create an HTML File

1. Open VS Code.
2. Create a new file.
3. Save the file with the **.html** extension.

Example:

```text
index.html
```

Any file ending with `.html` is treated as an HTML file.

# HTML Elements

HTML is made up of different elements. Each element has a specific purpose.

## Heading Tags

HTML provides six heading tags.

```html
            <h1>This is Heading 1</h1>
            <h2>This is Heading 2</h2>
            <h3>This is Heading 3</h3>
            <h4>This is Heading 4</h4>
            <h5>This is Heading 5</h5>
            <h6>This is Heading 6</h6>
```

* `<h1>` is the largest heading.
* `<h6>` is the smallest heading.

## Paragraph Tag

The `<p>` tag is used to write paragraphs.

```html
            <p>This is a paragraph.</p>
```

## Line Break Tag

The `<br>` tag is used to move content to the next line.

```html
            Hello<br>World
```

* `<br>` is an **empty element**.
* It does not have a closing tag.
* It does not contain any content.

# Running HTML Code

After writing your HTML code:

1. Install the **Live Server** extension in VS Code.
2. Open your HTML file.
3. Right-click inside the file.
4. Select **Open with Live Server**.
5. Your webpage will open in your default web browser.

# HTML Structure

A basic HTML page looks like this:

```html
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>
        <p>This is another paragraph.</p>
    </body>
</html>
```

Structure Explanation

    <!DOCTYPE html> – Declares the document as HTML5.
    <html> – Root element of the webpage.
    <head> – Contains page information.
    <title> – Sets the browser tab title.
    <body> – Contains all visible content.
    <!-- --> – Used for comments.

Common Tags

    <h1> to <h6> – Heading tags.
    <p> – Paragraph.
    <a href="URL">Text</a> – Creates a hyperlink.
    <img src="image.jpg" alt="Image"> – Displays an image.
    style="" – Adds inline CSS.
    lang="en" – Specifies the webpage language.
    title="" – Displays a tooltip on hover.
    <hr> – Horizontal line.
    <br> – Line break.
    <pre> – Preserves spaces and line breaks.

Text Formatting

    <b> – Bold
    <strong> – Important text
    <i> – Italic
    <em> – Emphasized
    <mark> – Highlighted
    <small> – Smaller text
    <del> – Deleted text
    <ins> – Inserted text
    <sub> – Subscript
    <sup> – Superscript

Quotation and Citation Elements
    <blockquote> - element defines a section that is quoted from another source.
    <q> - insert quotation marks around the quotation.
    <abbr> - tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
    <address> - tag defines the contact information for the author/owner of a document or an article.
    <cite> - tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).
    <bdo> -Bi Directional Override. 

HTML is the foundation of every website. It provides the structure, while CSS is used for styling and JavaScript is used to add interactivity.
