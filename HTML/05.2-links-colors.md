# HTML Link Colors

By default, browsers display links with different colors depending on their state. You can also customize these colors using CSS.

---

# Default Link Colors

Most browsers use the following default colors:

| Link State | Default Color | Description |
|------------|--------------|-------------|
| Unvisited Link | Blue | A link that has not been clicked yet |
| Visited Link | Purple | A link that has already been visited |
| Active Link | Red | A link while it is being clicked |

Example:

```html
<a href="https://www.google.com">
    Visit Google
</a>
```

---

# Changing Link Colors with CSS

You can change the color of links using CSS pseudo-classes.

```css
a:link {
    color: blue;
}

a:visited {
    color: purple;
}

a:hover {
    color: red;
}

a:active {
    color: green;
}
```

---

# Link States

There are four main link states.

## 1. `:link`

Represents a link that has **not been visited**.

Example:

```css
a:link {
    color: blue;
}
```

---

## 2. `:visited`

Represents a link that the user has already visited.

Example:

```css
a:visited {
    color: purple;
}
```

---

## 3. `:hover`

Represents a link when the mouse pointer is placed over it.

Example:

```css
a:hover {
    color: red;
}
```

You can also change other properties on hover.

Example:

```css
a:hover {
    color: red;
    text-decoration: underline;
}
```

---

## 4. `:active`

Represents a link while it is being clicked.

Example:

```css
a:active {
    color: green;
}
```

---

# Order of Link Styles

When using multiple link pseudo-classes, they should be written in this order:

```css
a:link
a:visited
a:hover
a:active
```

A simple way to remember this order is:

> **L V H A**  
> **L**ink → **V**isited → **H**over → **A**ctive

---

# Removing the Underline

By default, links are underlined.

To remove the underline:

```css
a {
    text-decoration: none;
}
```

To show the underline only when hovering:

```css
a {
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}
```

---

# Changing the Background Color

You can also change the background color of a link.

```css
a:hover {
    background-color: yellow;
}
```

---

# Styling Links as Buttons

Links can be styled to look like buttons using CSS.

```html
<a href="about.html" class="button">
    About Us
</a>
```

```css
.button {
    background-color: royalblue;
    color: white;
    padding: 10px 20px;
    text-decoration: none;
    border-radius: 5px;
}

.button:hover {
    background-color: dodgerblue;
}
```

---

# Complete Example

```html
<!DOCTYPE html>
<html>
<head>

<style>

a:link {
    color: blue;
}

a:visited {
    color: purple;
}

a:hover {
    color: red;
    text-decoration: underline;
}

a:active {
    color: green;
}

</style>

</head>

<body>

<h2>HTML Link Colors</h2>

<a href="https://www.google.com">
    Visit Google
</a>

</body>
</html>
```

---

# Summary

| Pseudo-class | Purpose |
|--------------|---------|
| `:link` | Styles an unvisited link |
| `:visited` | Styles a visited link |
| `:hover` | Styles a link when the mouse hovers over it |
| `:active` | Styles a link while it is being clicked |
| `text-decoration` | Adds or removes the underline |
| `background-color` | Changes the link's background color |
| `color` | Changes the text color of the link |

> **Note:** Link colors are controlled using **CSS**, not HTML. HTML creates the link using the `<a>` element, while CSS changes how the link looks.
