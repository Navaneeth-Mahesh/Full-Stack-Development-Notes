# HTML (HyperText Markup Language)

HTML stands for **HyperText Markup Language**. It is the standard markup language used to create and structure web pages.

HTML was invented by **Sir Tim Berners-Lee** and serves as the foundation of every website. It defines the structure and content of a webpage using **HTML elements**.

An HTML element tells the browser how different types of content should be displayed, such as:

- Headings
- Paragraphs
- Images
- Links
- Tables
- Lists
- Forms
- And many more

> **Note:** HTML provides the structure of a webpage, CSS is used for styling, and JavaScript is used to add interactivity.

---

## Basic HTML Example

```html
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>

<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>

<script>
    // JavaScript code can be written here
</script>

</html>
```

---

## How to Create an HTML File

1. Open **Visual Studio Code (VS Code)**.
2. Create a new file.
3. Save the file with the `.html` extension.

Example:

```text
index.html
```

Any file ending with the `.html` extension is recognized by the browser as an HTML document.
