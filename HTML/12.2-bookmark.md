# HTML Bookmarks

HTML bookmarks allow users to jump directly to a specific section of the same webpage or to a section in another webpage.

Bookmarks are useful for:

- Long articles
- Documentation
- FAQs
- Table of Contents
- Navigation menus

---

# How Bookmarks Work

Creating a bookmark involves two simple steps:

1. Add an `id` attribute to the element you want to jump to.
2. Create a link using `href="#id"`.

---

# Step 1: Create the Bookmark

Assign a unique `id` to an HTML element.

### Example

```html
<h2 id="about">About Us</h2>

<p>This section contains information about our company.</p>
```

Here, `about` is the bookmark name.

---

# Step 2: Create the Link

Use the `href` attribute with `#` followed by the `id`.

### Example

```html
<a href="#about">Go to About Section</a>
```

When the user clicks the link, the browser automatically scrolls to the element with `id="about"`.

---

# Complete Example

```html
<!DOCTYPE html>
<html>

<head>
    <title>HTML Bookmarks</title>
</head>

<body>

    <h1>HTML Bookmarks Example</h1>

    <a href="#services">Go to Services</a>

    <br><br>

    <a href="#contact">Go to Contact</a>

    <br><br>

    <p>
        Scroll down to see the bookmark in action.
    </p>

    <br><br><br><br><br><br><br><br><br><br>

</body>

</html>
```

# Linking to a Bookmark on Another Page

You can also jump to a bookmark on a different HTML page.

### Syntax

```html
<a href="about.html#team">
    Meet Our Team
</a>
```

Example:

```html
<!-- about.html -->

<h2 id="team">Our Team</h2>
```

Clicking the link opens `about.html` and scrolls directly to the **Our Team** section.

# Bookmark Navigation Example

```html
<h1>My Website</h1>

<a href="#home">Home</a> |
<a href="#about">About</a> |
<a href="#contact">Contact</a>

<hr>

<h2 id="home">Home</h2>

<p>Welcome to our website.</p>

<h2 id="about">About</h2>

<p>We build modern websites.</p>

<h2 id="contact">Contact</h2>

<p>Contact us at info@example.com</p>
```

# Best Practices

- Use meaningful and unique `id` values.
- Avoid spaces in `id` names.
- Keep bookmark names short and descriptive.
- Use bookmarks for long webpages to improve navigation.

### Good

```html
<h2 id="about-us">About Us</h2>

<a href="#about-us">About Us</a>
```

### Avoid

```html
<h2 id="section123">About Us</h2>
```

---

# Summary

| Feature | Description |
|---------|-------------|
| `id` | Creates a bookmark target |
| `href="#id"` | Links to a bookmark on the same page |
| `page.html#id` | Links to a bookmark on another page |
| Bookmarks | Help users quickly navigate long webpages |
