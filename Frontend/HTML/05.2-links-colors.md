# HTML Link Colors

By default, browsers display hyperlinks in different colors based on their current state (such as unvisited, visited, or active). You can customize these colors and styles using **CSS**.

> **Note:** HTML is used to create links with the `<a>` element, while **CSS** is used to change how those links look.

---

# Default Link Colors

Most browsers use these default colors for links:

| Link State | Default Color | Description |
|------------|--------------|-------------|
| Unvisited (`:link`) | Blue | A link that has not been clicked yet |
| Visited (`:visited`) | Purple | A link that has already been visited |
| Active (`:active`) | Red | A link while it is being clicked |

### Example

```html
<a href="https://www.google.com">
    Visit Google
</a>
```

---

# Understanding Link States

A hyperlink can be in four different states.

```text
            Mouse Over
      :link ----------> :hover
        |                  |
        | Click            | Mouse Leaves
        v                  |
     :active               |
        |                  |
        +------------------+
             After Visit
                  ↓
             :visited
```

---

# CSS Link Pseudo-Classes

CSS provides four pseudo-classes for styling links.

## 1. `:link`

Styles a hyperlink that **has not been visited**.

### Syntax

```css
a:link {
    color: blue;
}
```

---

## 2. `:visited`

Styles a hyperlink that the user **has already visited**.

### Syntax

```css
a:visited {
    color: purple;
}
```

> **Note:** Modern browsers restrict which CSS properties can be changed for visited links to protect user privacy. In most cases, only color-related properties are allowed.

---

## 3. `:hover`

Styles a hyperlink when the mouse pointer is placed over it.

### Syntax

```css
a:hover {
    color: red;
}
```

You can also change other CSS properties.

```css
a:hover {
    color: red;
    text-decoration: underline;
}
```

Common hover effects include:

- Changing text color
- Adding or removing underlines
- Changing the background color
- Adding borders
- Scaling or animating the element (using CSS)

---

## 4. `:active`

Styles a hyperlink **while it is being clicked**.

```css
a:active {
    color: green;
}
```

The active state usually lasts only while the mouse button is pressed.

---

# Styling All Link States Together

```css
a:link {
    color: blue;
}

a:visited {
    color: purple;
}

a:hover {
    color: red;
}

a:active {
    color: green;
}
```

---

# Correct Order of Link Styles

When writing multiple link pseudo-classes, use them in this order:

```css
a:link
a:visited
a:hover
a:active
```

Remember it as:

> **L V H A**

- **L** → Link
- **V** → Visited
- **H** → Hover
- **A** → Active

Using this order ensures the styles work correctly because CSS applies rules based on specificity and order.

---

# Removing the Underline

By default, hyperlinks are underlined.

## Remove the underline

```css
a {
    text-decoration: none;
}
```

Result:

```text
Before:
Visit Google

After:
Visit Google
```

---

# Show Underline Only on Hover

```css
a {
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}
```

This creates a cleaner design while still indicating that the text is clickable.

---

# Changing the Link Background

You can also change the background color.

```css
a:hover {
    background-color: yellow;
}
```

Example effect:

```text
Before Hover

Visit Google

After Hover

[ Visit Google ]
```

---

# Styling Links as Buttons

Since hyperlinks are inline elements, CSS can make them look like buttons.

### HTML

```html
<a href="about.html" class="button">
    About Us
</a>
```

### CSS

```css
.button {
    background-color: royalblue;
    color: white;
    padding: 10px 20px;
    text-decoration: none;
    border-radius: 5px;
}

.button:hover {
    background-color: dodgerblue;
}
```

Result:

```text
+----------------+
|   About Us     |
+----------------+
```

This technique is commonly used for:

- Navigation menus
- Download buttons
- Login buttons
- Call-to-action (CTA) buttons

---

# Complete Example

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML Link Colors</title>

    <style>
        a:link {
            color: blue;
        }

        a:visited {
            color: purple;
        }

        a:hover {
            color: red;
            text-decoration: underline;
        }

        a:active {
            color: green;
        }
    </style>
</head>

<body>

    <h2>HTML Link Colors</h2>

    <a href="https://www.google.com">
        Visit Google
    </a>

</body>
</html>
```

# Best Practices

- Keep link colors easy to read.
- Make sure links are visually different from normal text.
- Add hover effects to improve user experience.
- Maintain sufficient color contrast for accessibility.
- Use buttons (`<button>`) for actions and links (`<a>`) for navigation.

> **Rule of Thumb**
>
> - Use `<a>` when navigating to another page or website.
> - Use `<button>` when performing an action such as submitting a form or opening a dialog.

---

# Common Mistakes

### Incorrect

```css
a:hover {
    color: red;
}

a:visited {
    color: purple;
}
```

The order is incorrect.

### Correct

```css
a:link
a:visited
a:hover
a:active
```

---

### Incorrect

```html
<a href="#">Submit Form</a>
```

If the element is meant to submit a form, use a `<button>` instead.

# Summary

| Pseudo-class | Purpose |
|--------------|---------|
| `:link` | Styles an unvisited link |
| `:visited` | Styles a visited link |
| `:hover` | Styles a link when the mouse pointer is over it |
| `:active` | Styles a link while it is being clicked |
| `text-decoration` | Adds or removes the underline |
| `color` | Changes the text color |
| `background-color` | Changes the background color |

---

# Key Points

- HTML creates hyperlinks using the `<a>` element.
- CSS controls the appearance of hyperlinks.
- Every hyperlink can have four different states.
- Write pseudo-classes in the order **Link → Visited → Hover → Active (LVHA)**.
- Hover effects improve user experience.
- Links can be styled to look like buttons using CSS.
- Use links for navigation and buttons for actions.
