# JavaScript Introduction

# What is JavaScript?

JavaScript (JS) is a programming language used to make websites interactive and dynamic.

With JavaScript, you can:
- Respond to button clicks
- Show or hide content
- Validate forms
- Create animations
- Build games
- Fetch data from servers
- Develop full-stack applications

JavaScript works in all modern web browsers and can also run outside the browser using **Node.js**.

---

# Why Learn JavaScript?

JavaScript is one of the most popular programming languages because it allows you to build almost anything.

You can create:
- Websites
- Mobile apps
- Desktop applications
- Games
- AI-powered applications
- Dashboards
- Backend APIs

If you want to become a **Full Stack Developer**, learning JavaScript is essential.

# JavaScript in Web Development

A website is mainly built using three technologies:

| Technology    | Purpose |
|---------------|---------|
| HTML          | Creates the structure |
| CSS           | Adds styles and design |
| JavaScript    | Adds functionality and interactivity |

Example:

- HTML creates a button.
- CSS makes it look attractive.
- JavaScript makes it do something when clicked.

# JavaScript vs Java

Many beginners think JavaScript and Java are the same, but they are completely different languages.

| JavaScript                            | Java |
|---------------------------------------|------|
| Mainly used for web development       | Used for desktop, Android, and enterprise applications |
| Runs in browsers and Node.js          | Runs on the Java Virtual Machine (JVM) |
| Dynamically typed                     | Statically typed |
| Easier for beginners                  | More strict syntax |

**Remember:** JavaScript is **not** the same as Java.
